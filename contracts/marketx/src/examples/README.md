# MarketX Contract Frontend Integration Examples

This directory contains sample code for interacting with the MarketX smart contract using the Freighter wallet.

## Requirements
- Node.js (for running a local server)
- Freighter wallet extension installed
- Stellar testnet account with funds

## How to Run
1. Navigate to `contracts/marketx/examples/`.
2. Start a simple HTTP server:
   ```bash
   npx serve .
